WFPS Web Server

WFPS - Wireless Fire Protection System.

Specifications :

  • This system is designed wireless and has the ability to be accessed via the internet so that it can be opened from anywhere (as long as there is internet facilities).
  • Local panel can also be used for operational (viewing status, parameter settings change) use a keypad. Here is the LCD to display messages to the user.
  • EoN (Edge of Network) is the node closest to zone where sensors, solenoid valves, push buttons, LEDs are attached for protection purposes.
  • Communication protocol between broker (server) and the sensor / actuator uses MQTT (Message Queuing Telemetry Transport), for internet network uses HTTP (Hypertext Transfer Protocol).
  • This system uses OOP (Object Oriented Programming) approach and uses the MVC (Model-View-Controller) pattern.
  • This approach is taken to make it easier to develop and easy to track if there are problems.
